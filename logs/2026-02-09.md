# 2026/02/09 学習ログ

## 作業内容
- Packet Tracerで Router-on-a-Stick 構成を作成
- VLAN10 / VLAN20 を作成
- スイッチの access / trunk 設定
- ルータで subinterface 設定（dot1Q）
- PC間の疎通確認（ping 成功)

## 学んだこと
- Router-on-a-Stick はルータ1ポートでVLAN間ルーティングを行う
- 親インターフェースは IP を持たない（unset で正常）
- subinterface に VLAN番号と IP を設定する
- trunk はスイッチ側で設定する
- 物理接続ミスと設定ミスは切り分けて考える
