# 2026/01/23 学習ログ

## 目的
- Ubuntu Serverの初期構築を行う

## 作業内容
- UTM上にUbuntu Serverを初期構築
- SSHサーバー（OpenSSH）のインストール状態確認
- IPアドレスを確認しネットワークに接続を確認
- SSH接続試行
- SSHサービスが停止（inactive）していたため、起動および自動起動設定を実施
  sudo systemctl start ssh
  sudo systemctl enable ssh

## 学んだこと
- Ubuntu Server は
インストールされているだけでは利用できず、
必要なサービス（sshd）を起動して初めて外部接続が可能になること

- SSH接続は
インストール状態とサービスの起動状態の両方を確認する必要があること

- SSH鍵認証は
接続元（Mac）で秘密鍵公開鍵を作成し、接続先（Ubuntu）に公開鍵を登録するという仕組み
